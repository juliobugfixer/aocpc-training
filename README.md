## 📌 Descrição

Repositório dedicado ao meu treinamento para competições de programação, com foco no **AOCPC (Angola Online Competitive Programming Contest)**.

Aqui reúno soluções, anotações e práticas resolvidas em diferentes plataformas, com o objetivo de melhorar:

- 🧠 Raciocínio lógico  
- ⚡ Performance em algoritmos  
- 📊 Estruturas de dados  
- 🏆 Preparação para competições
